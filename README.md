# Gasdelivery.nl

Next.js (App Router), TypeScript, Tailwind CSS v4.

**Hosting:** push naar Git en koppel de repo aan [Vercel](https://vercel.com) — geen extra config nodig voor een standaard Next.js-app.

**Lokaal (optioneel):** `npm install` → `npm run dev` → [localhost:3000](http://localhost:3000).

**SEO:** kopieer `.env.example` naar `.env.local` en zet `NEXT_PUBLIC_SITE_URL` op je echte domein (bijv. `https://gasdelivery.nl`) vóór productie.
